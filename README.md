# 🎮 Tic Tac Toe Android Game

A simple and interactive **Tic Tac Toe game for Android** built using **Java and XML in Android Studio**. This project is designed for two players and includes winner detection, draw handling, and a custom winner screen using Intents.

---

## 📱 Features
- Added a animated loading screen
- 🎮 Two-player gameplay (X and O)
- 🏆 Winner detection (all rows, columns, diagonals)
- 🤝 Draw condition handling
- 🔁 Restart game functionality
- 🎨 Custom UI with styled buttons and rounded corners
- 📲 Separate Winner Activity screen using Intent
- ⏱ Smooth game flow with delayed restart logic

---

## 🛠️ Tech Stack

- Java (Android Development)
- XML (UI Design)
- Android Studio

## ⚙️ How It Works

1. Players take turns marking X and O.
2. After every move, the app checks all winning combinations:
   - Rows
   - Columns
   - Diagonals
3. If a player wins:
   - A Winner Activity opens showing the winner.
4. If all cells are filled:
   - The game ends in a draw screen.
5. Game can be restarted after result.
